什么是支持向量？

为什么需要核函数？

对于完全线性不可分的情况，我们可以将数据映射到高维，从而线性可分。回顾线性可分的SVM优化函数：
$$
\begin{gathered}\underbrace{\min }{\alpha} \frac{1}{2} \sum{i=1}^{m} \sum_{j=1}^{m} \alpha_{i} \alpha_{j} y_{i} y_{j}\left(x_{i} \cdot x_{j}\right)-\sum_{i=1}^{m} \alpha_{i} \text { s.t. } \sum_{i=1}^{m} \alpha_{i} y_{i}=0 \alpha_{i} \geq 0 i=1,2, \ldots m\end{gathered}
$$
上式低维特征仅仅以内积$𝑥𝑖∙𝑥𝑗\phi$
\begin{gathered}
\underbrace{\min }{\alpha} \frac{1}{2} \sum{i=1, j=1}^{m} \alpha_{i} \alpha_{j} y_{i} y_{j} \phi\left(x_{i}\right) \cdot \phi\left(x_{j}\right)-\sum_{i=1}^{m} \alpha_{i} \
\text { s.t. } \sum_{i=1}^{m} \alpha_{i} y_{i}=0 \
0 \leq \alpha_{i} \leq C
\end{gathered}
$$
但是这样我们需要求出内积，这样类似于二次规划那样会引入一个维空间，从而发生维度爆炸影响计算速度。

什么是软间隔？

前面讲到线性SVM一个关键点就是超平面必须完全分类所有点，但实际上一方面有些数据混入了异常点，导致本来线性可分变成了不可分，如下图一个橙色和蓝色的异常点导致我们无法用线性SVM。

但还有一种情况是没有糟糕到那么不可分，如下图，本来如果不考虑异常点，SVM的超平面应该是如红色线所示，但由于有一个蓝色的异常点，导致我们学到的超平面是如粗虚线所示。

为解决这个问题，SVM引入了软间隔的方法。

下面我们需要优化软间隔SVM优化函数，与线性SVM类似，引入拉格朗日乘子转为无约束问题：
$$
L(w, b, \xi, \alpha, \mu)=\frac{1}{2}|w|{2}^{2}+C \sum{i=1}^{m} \xi_{i}-\sum_{i=1}^{m} \alpha_{i}\left[y_{i}\left(w^{T} x_{i}+b\right)-1+\xi_{i}\right]-\sum_{i=1}^{m} \mu_{i} \xi_{i}

\underbrace{\max }{\alpha{i} \geq 0, \mu_{i} \geq 0} \underbrace{\min }{w, b, \xi} L(w, b, \alpha, \xi, \mu)

\begin{gathered}
\frac{\partial L}{\partial w}=0 \Rightarrow w=\sum{i=1}^{m} \alpha_{i} y_{i} x_{i} \
\frac{\partial L}{\partial b}=0 \Rightarrow \sum_{i=1}^{m} \alpha_{i} y_{i}=0 \
\frac{\partial L}{\partial \xi}=0 \Rightarrow C-\alpha_{i}-\mu_{i}=0
\end{gathered}
$$

最后导出软间隔SVM的优化函数为:
$$
\begin{gathered}
\underbrace{\min }{\alpha} \frac{1}{2} \sum{i=1, j=1}^{m} \alpha_{i} \alpha_{j} y_{i} y_{j} x_{i}^{T} x_{j}-\sum_{i=1}^{m} \alpha_{i} \
\text { s.t. } \sum_{i=1}^{m} \alpha_{i} y_{i}=0 \
0 \leq \alpha_{i} \leq C
\end{gathered}
$$
可以看到与硬间隔SVM相比，只是多了一个约束条件。

为什么用拉格朗日对偶去解决优化问题？而不是直接使用现成的优化计算包？

开始说了， 目标是降低算法的规模或者说是复杂度。原始问题的最关于后需要解决的minb,w部分的复杂度是关于w的，也就是高维空间的维度，求解很困难。而对偶问题最后要解决的maxα则是关于αn的，复杂度跟样本数量N有关，在非线性分类中一般情况升维会达到很大的数量级(比如后续的高斯核是无穷维)，远远大于样本数量，这个时候用对偶问题来求解就会容易。

此外，如果是线性分类且样本维度低于样本数量的话，那就直接使用原始问题求解即可，无需对偶。

为什么要交换min max后再进行计算？如若不然？

我们通常会将线性SVM转化为对偶问题，主要解决在非线性转换之后维度变高二次规划问题求解困难的问题。在非线性分类问题中，我们一般作非线性转换$\mathbf{z}{n}=\phi\left(\mathbf{x}{n}\right)\mathbf{z}\mathbf{z}\tilde{d}\tilde{d}>>d\tilde{d}+1\tilde{d}+1mQ\tilde{d}^2\tilde{d}$变大了， 二次规划问题求解就会变得很困难。

我们引入拉格朗日乘子来转化为对偶问题，优化函数变为：

优化目标为：
$$
\underbrace{\min }{w, b} \underbrace{\max }{\alpha_{i} \geq 0} L(w, b, \alpha)
$$
拉格朗日对偶的关键优势在于它将原始问题分解为两个相对容易解决的问题：一个是最大化对偶问题，另一个是使用对偶问题的解来计算原始问题的最小值

由于这个优化函数满足KTT条件，因此通过拉格朗日对偶将优化目标转为：
$$
\underbrace{\max }{\alpha{i} \geq 0} \underbrace{\min }_{w, b} L(w, b, \alpha)
$$
根据上式，我们可以先求出优化函数对于的极小值，然后求拉格朗日乘子的最大值。

我们先求出优化函数对于的极小值，即$\underbrace{\min }{w, b} L(w, b, \alpha)w,b$
\begin{gathered}
\frac{\partial L}{\partial w}=0 \Rightarrow w=\sum{i=1}^{m} \alpha_{i} y_{i} x_{i} \
\frac{\partial L}{\partial b}=0 \Rightarrow \sum_{i=1}^{m} \alpha_{i} y_{i}=0
\end{gathered}
$$
这样我们就知道了和的关系，然后代入$\underbrace{\min }{w, b} L(w, b, \alpha)w$
\psi(\alpha)=\underbrace{\min }{w, b} L(w, b, \alpha)

\underbrace{\max }{\alpha}-\frac{1}{2} \sum{i=1}^{m} \sum_{j=1}^{m} \alpha_{i} \alpha_{j} y_{i} y_{j}\left(x_{i} \cdot x_{j}\right)+\sum_{i=1}^{m} \alpha_{i} \
\text { s.t. } \sum_{i=1}^{m} \alpha_{i} y_{i}=0 \
\alpha_{i} \geq 0 i=1,2, \ldots m
\end{gathered}

\text { s.t. } \sum_{i=1}^{m} \alpha_{i} y_{i}=0 \
